# AI Research Assistant

A clean Python-based AI Research Assistant built using Streamlit.

## Project Structure
- `app.py`: The main entry point for the Streamlit web application.
- `pdf_processor.py`: Module containing utility functions and classes for processing and extracting content from PDF documents.
- `ai_service.py`: Service module to handle communication and reasoning tasks using AI models.
- `utils.py`: Utility helper functions for layout, styling, and general operations.
- `requirements.txt`: Defines the Python package dependencies for the project.
- `.env.example`: Example environment file containing template keys for configuration.
- `README.md`: Documentation containing a project overview and instructions.
- `assets/`: Folder for static assets (images, icons, styles).
- `screenshots/`: Folder for screenshots demonstrating project UI and usage.
